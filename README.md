# QNA Consulting

**A division of QNA Holdings** — Engineering consulting at the intersection of broadcast, cloud, and infrastructure.

## About

QNA Consulting delivers technical consulting services across broadcast engineering, cloud infrastructure, networking, Linux/server OS, and software development. Projects range from greenfield builds to modernizing legacy systems for media, property management, and enterprise clients.

| Domain | Detail |

| **Cloud Infrastructure** | AWS architecture, serverless design, cost optimization, IaC |

| **Linux / Server OS** | System administration, hardening, automation, performance tuning |
| **Software Development** | Backend services, APIs, CLI tooling, infrastructure-adjacent software |
